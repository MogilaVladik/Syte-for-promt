# ⚠️ ВАЖНО: Установка зависимостей

## Проблема с типами TypeScript

Ошибки линтера появляются потому, что **не установлены зависимости проекта** (отсутствует папка `node_modules`).

## Решение

### 1. Установите зависимости

Выберите один из вариантов в зависимости от того, какой менеджер пакетов установлен:

```bash
# Вариант 1: npm
npm install

# Вариант 2: yarn  
yarn install

# Вариант 3: bun (рекомендуется для этого проекта)
bun install
```

### 2. Перезапустите TypeScript Server

После установки зависимостей:

1. Откройте Command Palette (`Cmd+Shift+P` в Mac, `Ctrl+Shift+P` в Windows/Linux)
2. Выберите: **"TypeScript: Restart TS Server"**

### 3. Проверьте работу

```bash
npm run dev
# или
bun run dev
```

## Почему возникают ошибки?

TypeScript не может найти типы React (`@types/react`, `@types/react-dom`), потому что они находятся в `node_modules`, который не установлен.

После установки зависимостей все ошибки типов исчезнут автоматически.

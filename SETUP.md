# Инструкция по установке и решению проблем

## Установка зависимостей

Для работы проекта необходимо установить зависимости. Выберите один из вариантов:

### Вариант 1: Используя npm
```bash
npm install
```

### Вариант 2: Используя yarn
```bash
yarn install
```

### Вариант 3: Используя bun (рекомендуется для этого проекта)
```bash
bun install
```

## После установки зависимостей

1. **Перезапустите TypeScript Server в редакторе:**
   - В VS Code/Cursor: `Cmd+Shift+P` → "TypeScript: Restart TS Server"

2. **Очистите кэш сборки:**
   ```bash
   rm -rf .next
   rm -rf node_modules/.cache
   ```

3. **Пересоберите проект:**
   ```bash
   npm run build
   # или
   bun run build
   ```

## Решение проблем с типами

Если после установки зависимостей всё ещё есть ошибки типов:

1. Убедитесь, что установлены все зависимости:
   ```bash
   npm list @types/react @types/react-dom
   ```

2. Проверьте версии в `package.json`:
   - `react`: ^19.2.0
   - `@types/react`: ^19
   - `@types/react-dom`: ^19

3. Перезапустите редактор полностью

## Проверка работоспособности

После установки зависимостей запустите:
```bash
npm run dev
# или
bun run dev
```

Проект должен запуститься без ошибок типов.
